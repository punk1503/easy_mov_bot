# Easy Mov Bot
Слегка недописанный телеграм бот для конвертации видео в mp4. Требует установки ffmpeg в окружении.

Чтобы запустить, нужно прописать TOKEN = 'your_telegram_token' в файле config.py.

### Зависимости
- aiogram 3.x
- MoviePy
